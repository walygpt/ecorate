# EcoRate AI - Intelligent ESG Loan Management Platform

EcoRate AI is a sophisticated fintech dashboard designed to automate the relationship between corporate sustainability performance and loan interest rates. Built for the LMA EDGE Hackathon, this platform leverages AI to provide real-time ESG risk assessment and dynamic rate adjustments.

##  Key Features
- **AI-Driven ESG Audit:** Uses Google Gemini 1.5 Flash to analyze complex sustainability reports instantly.
- **Dynamic Interest Rate Rachets:** Automatically calculates basis point adjustments based on real-time ESG scores.
- **Enterprise Dashboard:** A professional "Bloomberg-style" UI for monitoring portfolio-wide green compliance.
- **PWA Ready:** Fully responsive and installable on mobile and desktop devices.

##  Tech Stack
- **Framework:** Next.js 16 (App Router)
- **Library:** React 19.2
- **Styling:** Tailwind CSS v4
- **AI Engine:** Google Gemini API
- **Deployment:** Vercel
- **Language:** TypeScript

##  Getting Started
1. Clone the repository: `git clone https://github.com/walygpt/ecorate.git`
2. Install dependencies: `pnpm install`
3. Set up your environment variables: Add `NEXT_PUBLIC_GEMINI_API_KEY` to your `.env` file.
4. Run the development server: `pnpm dev`
